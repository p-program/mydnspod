## 实现细节

一开始做的时候有点摸不着头脑,document经常获取不到元素.后来仔细观察发现这个破网站一开始用了 iframe 加载,过一段时间后又把iframe的标记去掉,于是我只能写个函数,监听这种变化.

除此以外,要考虑几个点:

1. 静态页面布局的调整
2. 对点击/修改/取消操作做了事件监听,最大程度对齐
